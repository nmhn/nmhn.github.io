第三步,下载对应数据库版本的jdbc,我的腾讯云是mysql8

jdbc位置是

MySQL :: MySQL Connectors

打开之后长这样,下载JDBC

这儿选platform independent,然后点第二个下载

  1. 把上面那玩意引入android
